Function CaptureWindow(ByVal hwnd As Long, ByVal destination As String, Optional ByVal Client As Boolean = False, Optional ByVal Text As String = "", Optional ByVal VerticalScroll As Boolean = True, Optional ByVal HorizontalScroll As Boolean = True)

Description
Captures the window with given handle.

Arguments
hwnd
Handle of the window
destination
The destination file name or special Destination Keyword where the screen shot has to be stored.
Client
Optional. Default Value: False. If set to True then the screen shot will not include the title bar of the window
Text
Optional. Default Value: "". Text to be added to screen shot. The position, height, font and font size of the same can be changed by changing the properties TextPos, TextHeight, TextFontName and TextFontSize respectively.
VerticalScroll
Optional. Default Value: True. Not implemented/Reserved for future use.
HorizontalScroll
Optional. Default Value: True. Not implemented/Reserved for future use.

Example
Capturing all open notepad windows
