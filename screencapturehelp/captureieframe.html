Function CaptureIEFrames(ByVal source As Object, ByVal destination As String, Optional ByVal Text As String = "", Optional ByVal VerticalScroll As Boolean = False, Optional ByVal HorizontalScroll As Boolean = False)

Description
Captures a web page open in internet explorer. Allows Horizontal as well as Vertical scroll capturing invisible are of web pages.

Arguments
source
Internet explorer object for which the object has to be capture. This can be a object of type "InternetExplorer.Application", QTP Browser object, QTP Page object, handle of the browser window or any control inside the browser.

destination
The destination file name or special Destination Keyword where the screen shot has to be stored.

Text
Optional. Default Value: "". Text to be added to screen shot. The position, height, font and font size of the same can be changed by changing the properties TextPos, TextHeight, TextFontName and TextFontSize respectively.

VerticalScroll
Optional. Default Value: False. Scroll page vertically in case the height of the page is larger then the window height.

HorizontalScroll
Optional. Default Value: False. Scroll page vertically in case the width of the page is larger then the window width.
