Function CaptureDesktopRect(ByVal x As Long, ByVal y As Long, ByVal width As Long, ByVal height As Long, ByVal destination As String, Optional ByVal text As String = "")

Description
Captures a specified rectangular area of the screen

Arguments
x
x co-ordinate of starting point of the region. The value is respective to the window and not to the screen. 0 would mean starting of the window
y
y co-ordinate of starting point of the region. The value is respective to the window and not to the screen. 0 would mean starting of the window
width
Width of the region to be captured
height
Height of the region to be captured
destination
The destination file name or special Destination Keyword where the screen shot has to be stored.
Text
Optional. Default Value: "". Text to be added to screen shot. The position, height, font and font size of the same can be changed by changing the properties TextPos, TextHeight, TextFontName and TextFontSize respectively.

Example
Capture rectangular area of desktop
