Function GetIEObjectFromProperty(ByVal PropName As String, ByVal PropValue As Object) As Object

Description
Get object reference to a IE window using the any property of the IE window or the document

Arguments
PropName
Name of the property to be used for searching.
PropValue
Value of the property to be use. This can be a object as well.


Examples
Get IE object from any property
