Function EnumIEFramesDocument(ByVal wb As Object) As Object

Description
Get document objects for all frames present in the web page. IE cross domain security blocks access of any IFRAME which loads a page from a domain different than main web page. This method would return the document object for the all the embedded FRAME and IFRAME tags

Arguments
wb
Main HTML document object.

Examples
Enumerate frames document on a web page
