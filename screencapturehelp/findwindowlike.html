Function FindWindowLike(Optional ByVal WindowTitle As String = "*", Optional ByVal WindowClass As String = "*")

Description
Find details of window (hwnd, ClassName and Title) using it's title and/or class.

Arguments
WindowTitle
Optional. Default Value: "*". Title of the window to search for. Can take regular expression values. To search for all window with title containing "notepad" use "*notepad*" as window title.
WindowClass
Optional. Default Value: "*". Class of the window to search for. Can take regular expression values.

Returns
Collection of WindowInfo object.

Example
Capturing all open notepad windows
