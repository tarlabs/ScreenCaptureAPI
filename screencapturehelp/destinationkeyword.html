Destination Keywords

The following special destination keywords can be used with following functions
CaptureActiveWindow
CaptureDesktop
CaptureIE
CaptureScreenRect
CaptureWindow
CaptureWindowRect
CompareImages

[Clipboard]
Copies the captured image to clipboard instead of saving it.
Capture and copy image to clipboard

[User]
This keyword should be used when you want the user to select the destination file name. Using this keyword will display a Save As dialog.
Saving an image at a user selected location

[Object]
Returns the .NET Bitmap object for the captured image. This is for advanced users only and should be used with care. If not properly used this can cause memory leakage in programs.
Capture and combine images into one


The following destination keywords can only be used with CompareImages

[PixelDiffCount]
Returns the numbers of pixel which are different in both images being compared
Compare two images

[PixelDiffPerc]
Returns the percentage of pixels which are different in both images being compared
Compare two images
