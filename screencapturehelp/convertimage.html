Sub ConvertImage(ByVal sourceFile As String, ByVal destFile As String, Optional ByVal OverWriteDestination As Boolean = True)

Description
Converts file from one format to another format

Arguments
sourceFile
Source image file which needs to be converted
destFile
Path of the destination file where the file has to be stored. The extension of the file determine the new file format. The file formats supported for destination files are (png, bmp, jpg, tiff, , wmf, emf, EXIF, gif)
OverWriteDestination
Optional. Default Value: True. If the destination file exists and the value is set to False then a exception is raised.

Example
Convert image from one format to another
