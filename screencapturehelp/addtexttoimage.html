Function AddTextToImage(ByVal oImage As Bitmap, ByVal Text As String)

Description
Adds Text to a .NET Bitmap object

Arguments
      oImage
            Image object of type Bitmap. This is a .NET bitmap object.
Text
Text to be added to screen shot. The position, height, font and font size of the same can be changed by changing the properties TextPos, TextHeight, TextFontName and TextFontSize respectively.
